## brew-head

This tool helps the management of [brew] packages installed with the `--HEAD` option.

Type `brew help head` to get its usage page.

This tool depends on:

- [bash]
- [brew]
- [jq]
- [xargs]

[bash]: https://www.gnu.org/software/bash/
[brew]: https://brew.sh/index.html
[xargs]: http://pubs.opengroup.org/onlinepubs/9699919799/utilities/xargs.html
